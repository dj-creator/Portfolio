Spooky Doors is a game that I created using JavaScript and CSS. The game appears to be decision-based, which means players have to make choices that will impact the outcome of the game. Additionally, the game has jump scares, which adds an element of surprise and horror to the gameplay experience. It is an exciting game that keeps players on the edge of their seats as they make decisions and try to navigate through the spooky doors.
